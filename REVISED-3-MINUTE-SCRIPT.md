## Opik AI System Explained

**Architecture:**
```
Player (You)
    ↓ Complete Quest
System (Groq AI)
    ↓ Generate & Evaluate
Judge (Opik AI)
    ↓ Verify & Approve
User (You)
```

**Key Points:**
- Groq generates personalized quests and evaluates performance
- Opik acts as judge, evaluating Groq's output (not images directly)
- Opik ensures fair scoring before results reach user
- Provides personalized coaching and real-time feedback
- Full transparency - all judgments traced and logged
- No cheating possible with this three-layer system
