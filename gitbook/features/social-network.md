# Social Network

Connect with other hunters through the Hunter Network.

## Hunter Network

The social feed that shows quest completions across the community.

### Feed Types

| Type | Description |
| ---- | ----------- |
| **Global Feed** | All public completions |
| **Following** | Completions from followed users |
| **Rank Feed** | Users within your rank |
| **Class Feed** | Users within your class |

## Interactions

### Kudos

Show appreciation for other hunters' achievements:

| Action | XP Reward |
| ------ | --------- |
| Give kudos | 0 XP |
| Receive kudos | +10-50 XP |

### Respects

High-level recognition for exceptional performance:

| Condition | XP Reward |
| --------- | --------- |
| S-grade quest completion | +25 XP (respects received) |
| A-grade quest completion | +15 XP (respects received) |

### Following

| Action | Limit |
| ------ | ----- |
| Follow users | Unlimited |
| Receive notifications | Quest completions |

## Privacy Settings

| Setting | Default | Description |
| ------- | ------- | ----------- |
| Public profile | On | Others can see profile |
| Public completions | On | Others see your completions |
| Public stats | On | Others see XP/rank |
| Show on leaderboards | On | Appear in rankings |

## Social Features

| Feature | Description |
| -------- | ----------- |
| **User Profiles** | View hunter stats and history |
| **Quest History** | Browse past completions |
| **Badges** | Display achievements |
| **Streaks** | Show consecutive days |
| **Comments** | Discuss completions |

## Community Guidelines

| Guideline | Description |
| ---------- | ----------- |
| Be respectful | Positive interactions only |
| No spam | Meaningful kudos and comments |
| Honest reporting | Only report suspicious activity |
| Support others | Encourage fellow hunters |

## Social Leaderboards

| Leaderboard | Ranked By | Update Frequency |
| ----------- | --------- | ---------------- |
| Global | Total XP | Real-time |
| Weekly | Weekly XP | Daily reset |
| Most Kudos | Kudos received | Real-time |
| Streaks | Consecutive days | Real-time |
