## How to Add Images

1. Create or capture images
2. Place them in this `.gitbook/assets/` folder
3. Ensure filenames match the references in markdown files
4. Images will automatically appear in GitBook when published

## Image Guidelines

| Guideline | Recommendation |
| ---------- | -------------- |
| **Format** | PNG or JPG |
| **Width** | 1200-1600 pixels |
| **Height** | 600-800 pixels (for hero images) |
| **Size** | Under 500KB |
| **Style** | Clean, professional, on-brand |
