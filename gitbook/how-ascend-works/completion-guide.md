# Completion Guide

How to successfully complete quests and maximize XP rewards.

## Completion Checklist

Before submitting your quest:

| Item | Required | Notes |
| ---- | -------- | ----- |
| All exercises completed | ✅ Yes | Follow rep/set counts |
| Proof uploaded | ✅ Yes | Photo or video |
| Good quality proof | ✅ Yes | Clear, well-lit |
| Within time limit | ✅ Yes | 15-90 min range |

## Maximizing XP

### Form Quality (40% of score)

| Tip | Impact |
| --- | ------ |
| Full range of motion | +0.10-0.15 |
| Proper tempo | +0.05-0.10 |
| Correct posture | +0.10-0.15 |
| Joint alignment | +0.05-0.10 |

### Effort Level (30% of score)

| Indicator | High Effort |
| --------- | ----------- |
| Breathing | Heavy, rhythmic |
| Sweating | Visible |
| Movement speed | Controlled intensity |
| Fatigue | Noticeable by end |

### Consistency (30% of score)

| Factor | Consistent |
| ------ | ---------- |
| Rep to rep | Same form each time |
| Set to set | Same effort each set |
| Timing | Steady pace |
| Rest periods | Adhered to |

## Common Mistakes

| Mistake | Impact | Fix |
| ------- | ------ | ---- |
| Rushing through | -0.20 to score | Focus on form |
| Skipping rest | Consistency penalty | Time rest periods |
| Poor video quality | Evaluation fails | Use good lighting |
| Incomplete reps | Form penalty | Full range of motion |

## Scoring Rubric

| Score Range | Grade | XP Multiplier |
| ----------- | ----- | ------------- |
| 0.90-1.00 | S | 1.5x |
| 0.80-0.89 | A | 1.3x |
| 0.70-0.79 | B | 1.1x |
| 0.60-0.69 | C | 1.0x |
| 0.50-0.59 | D | 0.9x |
| 0.00-0.49 | E | 0.8x |

## After Submission

| Event | Timing |
| ------ | ------ |
| Upload complete | Instant |
| Evaluation started | 5-10 seconds |
| Scores calculated | 10-20 seconds |
| XP awarded | Instant |
| Leaderboard updated | Instant |

## Retrying Quests

- No penalty for retrying
- Each attempt evaluated separately
- Best XP awarded (not cumulative)
