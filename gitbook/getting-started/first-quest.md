## Step-by-Step

### Step 1: Generate Quest

1. Navigate to **"Quests"** page
2. Click **"Generate New Quest"**
3. Configure settings:
   - **Duration:** 30 minutes (recommended for first quest)
   - **Equipment:** Bodyweight (easiest)
   - **Goals:** Leave blank for now
4. Click **"Generate Quest"**

**Result:** You'll see your first quest with 4-6 exercises.

### Step 2: Review Your Quest

| Section | Description |
| -------- | ----------- |
| Quest Info | Rank, class, duration, XP |
| Exercises | List of exercises with reps/sets |
| Instructions | How to perform each exercise |

### Step 3: Complete the Workout

1. Start with exercise 1
2. Follow the rep count
3. Complete all sets
4. Rest between sets (30-90 seconds)
5. Move to next exercise
6. Repeat until complete

**Example Quest (C-Rank Striker):**

| Exercise | Sets | Reps |
| -------- | ---- | ---- |
| Jumping Jacks | 2 | 30 |
| Bodyweight Squats | 3 | 15 |
| Push-ups | 3 | 10 |
| Mountain Climbers | 3 | 20 |

### Step 4: Upload Proof

1. Click **"Complete Quest"**
2. Upload proof (photo or video)
3. Add optional notes
4. Click **"Submit for Evaluation"**

**Proof Tips:**
- Show your face and full body
- Capture at least 30 seconds
- Good lighting helps evaluation

### Step 5: Wait for Evaluation

| Phase | Duration | Description |
| ----- | -------- | ----------- |
| Upload | Immediate | Proof submitted |
| Processing | 10-30 seconds | OPIK AI analyzes |
| Evaluation | 5-10 seconds | Scores calculated |
| Results | Instant | XP awarded |

### Step 6: View Results

After evaluation, you'll see:

| Metric | Example |
| ------- | ------- |
| Form Score | 0.75 |
| Effort Score | 0.80 |
| Consistency Score | 0.72 |
| Overall Score | 0.76 (B Grade) |
| Multiplier | 1.1x |
| Base XP | 300 |
| Final XP | 330 |
