### Tools & Resources

**Video Editing:**
- DaVinci Resolve (Free, professional)
- CapCut (Mobile-friendly, Gen Z aesthetic)
- Premiere Pro (Professional option)

**Assets Needed:**
- ASCEND app footage (screen recordings)
- Anime-style transitions (preset packs)
- Epic music (royalty-free: Epidemic Sound, Artlist)
- "System" UI overlay elements (after effects or motion graphics)
- Stock footage (gym, anime conventions, gaming)

**Talent (Optional):**

- On-screen talent: Authentic users, not actors

**Budget:**
- DIY approach: $0-100 (stock music, footage)
- Pro approach: $500-2,000 (hiring editor, talent)
- Recommended: Start DIY, upgrade if time allows

### Production Timeline

**Day 1-2: Script & Storyboard (4-6 hours)**
- Write 60-90 second script
- Create visual storyboard (shot by shot)
- Identify assets needed (footage, music, graphics)

**Day 3-4: Footage Capture (6-8 hours)**
- Screen record app features
- Capture user interactions (multiple angles)
- Record demo account walkthrough
- B-roll footage (gym, anime conventions, community)

**Day 5-7: Editing (12-16 hours)**
- Rough cut: Assemble footage to script
- Fine cut: Refine timing, add transitions
- Audio: Add music, sound effects, voiceover
- Graphics: Add text overlays, UI elements
- Color grade: Match anime aesthetic

**Day 8-10: Polish (6-8 hours)**
- Review: Get feedback from target audience
- Adjust: Fix pacing, add jokes/memes
- Export: Multiple versions (60s, 90s, with/without music)
- Optimize: Compress for YouTube, embed in deck

**Total Time:** 28-38 hours over 10 days
