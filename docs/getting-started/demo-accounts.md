## Usage Guidelines

### Best Practices

1. **Choose Appropriate Rank**
   - Don't jump to S-Rank immediately
   - Start with C or B-Rank to understand progression
   - Experience unlocking features as you level up

2. **Experience Progression**
   - Try C → B → A → S to see features unlock
   - Notice how quest difficulty scales
   - Understand how XP requirements increase

3. **Test Different Classes**
   - Each class has unique quest types
   - Tank: Strength-focused workouts
   - Striker: Speed and cardio quests
   - Assassin: Agility and HIIT quests

4. **Explore Social Features**
   - Try Hunter Network feed
   - Give kudos/respects to other hunters
   - Check leaderboard rankings

### Account Limitations

- Demo accounts reset weekly
- No real personal data
- Can't earn real achievements
- Publicly accessible (use for testing only)
