## ğŸ¬ PRODUCTION TIMELINE

### Pre-Production (2-3 days)

**Day 1: Script & Storyboard (4-6 hours)**
- Write 180-second script (300-450 words)
- Create visual storyboard (shot-by-shot)
- Identify all footage needed
- List all assets (music, graphics, voiceover)

**Day 2: Footage Capture (6-8 hours)**
- Screen record all app features
- Capture demo account walkthrough
- Record multiple takes of each section
- B-roll footage (gym, community, anime conventions)

**Day 3: Asset Preparation (2-4 hours)**
- Select and license music
- Prepare UI overlays and graphics
- Record voiceover (or prepare for AI voice)
- Organize all footage by section

### Production (5-7 days)

**Day 4-5: Editing Phase 1 - Rough Cut (8-12 hours)**
- Assemble footage to match script
- Time each section to 3-minute limit
- Add basic transitions
- Sync voiceover to visuals

**Day 6-7: Editing Phase 2 - Polish (8-12 hours)**
- Add music and sound effects
- Insert text overlays and graphics
- Refine pacing (trim to exactly 3:00)
- Color grade for anime aesthetic

**Day 8-9: Editing Phase 3 - Final Polish (4-6 hours)**
- Review for timing (watch clock)
- Test on different devices
- Optimize file size for upload
- Export in required format

### Post-Production (1-2 days)

**Day 10: Final Review (2-4 hours)**
- Get feedback from target audience
- Make final adjustments
- Prepare backup versions
- Write description and tags

**Day 11: Submission (1-2 hours)**
- Upload to hackathon dashboard
- Add description and tags
- Verify file plays correctly
- Confirm submission complete
